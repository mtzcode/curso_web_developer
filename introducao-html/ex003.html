Título H1

Usado para o título principal da página, geralmente aparece uma vez por página para SEO e semântica.

Título H2

Usado para subtítulos principais, como seções importantes da página.

Título H3

Usado para subtítulos dentro de seções, como tópicos específicos.

Título H4

Usado para subtítulos de menor importância, como detalhes dentro de tópicos.

Título H5

Usado para subtítulos ainda menos relevantes, raramente utilizado.

Título H6

Usado para o menor nível de título, quase nunca usado em páginas comuns.
